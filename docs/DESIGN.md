# meteor-city — 디자인/아트 디렉션 브리프

> 이건 UI 디자인이 아니라 **3D 씬의 아트 디렉션**이다. developer 는 이 문서를 계약으로 R3F/three/postprocessing 을 세팅한다.
> 목표 한 줄: **"책상 위 정교한 도시 모형을 손끝으로 톡 — 그 순간 번쩍."** 캡처 한 장이 "우와 미니어처다" 로 읽혀야 성공.
> 수치는 전부 **시작점(seed value)**이다. developer 가 HDRI/GPU 따라 ±20% 튜닝하되, **비율·관계·톤 방향은 유지**한다.

---

## 0. 톤 & 무드

- 형용사: **정교한(crafted) · 아늑한 미니어처(toy diorama) · 절제된 고급(restrained premium) · 임팩트의 카타르시스**
- 첫인상 한 줄: 접속하면 이미 지어진 미니어처 도시가 3/4 부감으로 떠 있고, 위/아래가 살짝 흐려(틸트시프트) "진짜 모형 같다"는 착시가 먼저 온다. 화려함이 아니라 **정교함**으로 승부한다.
- 미니어처 룩의 3대 착시 축(이걸 다 만족해야 "장난감"으로 읽힌다):
  1. **틸트시프트 DOF** — 화면 상/하단 강한 블러, 중앙에 샤프 밴드.
  2. **살짝 올린 채도·대비 + 딱딱한 접지 그림자(AO/contact shadow)** — 실제 미니어처 사진의 특징. *과채도 아님*(§7).
  3. **유한한 디오라마 베이스** — 도시가 무한 지면이 아니라 **가장자리가 있는 모형판** 위에 놓였고, 그 가장자리가 안개/DOF 로 배경에 녹는다.

## 1. 시그니처 룩 (확정)

### 톤 결정: **"맑은 오후, 웜-뉴트럴 대낮 미니어처"** — 이걸 MVP 유일 톤으로 확정.
- 이유: 골든아워/야간 네온은 예쁘지만 **AI-에픽 클리셰**로 흐르기 쉽고(오렌지 떡칠·과한 발광), 색 판독이 어렵다. **밝은 대낮**이 틸트시프트 장난감 사진의 정석이고, **드라마는 임팩트 섬광(Bloom)이 담당**하므로 베이스 씬은 밝고 깨끗·중립이어야 섬광이 튄다.
- 태양 고도 **~40°**(정오 아님, 오후) → 형태를 살리는 **읽히는 길이의 부드러운 그림자**. 오렌지로 안 가는 선(색온도 ~5400K).
- (스트레치, MVP 제외) `무드=dusk` 토글 — 창문 emissive↑ + 하늘 그라디언트 웜다운. 지금은 만들지 말고 그레이딩만 스왑 가능하게 값 분리해 둘 것.

### 시그니처 이펙트: **TiltShift(가로 밴드 블러)** 를 이 앱의 서명으로 확정.
- `@react-three/postprocessing` 의 **`<TiltShift2>`**(스크린스페이스 가로 그라디언트 블러)를 1순위로 쓴다. 원형 DOF 보다 **진짜 틸트시프트 밴드**라 미니어처 착시가 강하고 저렴하다.
- 샤프 밴드는 화면 **세로 0.5~0.6 지점**(3/4 부감이라 도시가 중앙-약간아래에 앉음)에 두고, **조준/임팩트 지점으로 밴드 중심이 살짝 이동**(easing)해 액션은 늘 선명하게.

### 강물 (미니어처의 보석 — 조금 더 신경)
- drei **`MeshReflectorMaterial`** 권장(블러 반사로 고급감 저렴하게).
- 값: `color=#2E4A54`, `roughness=0.08`, `metalness=0`, `blur=[200, 60]`(살짝만), `mixStrength=0.55`, `depthScale=6`(가장자리 페이드), `minDepthThreshold=0.9`, 미세 노멀맵 리플(scale 0.02). 반사에 하늘+건물이 비쳐야 "물" 로 읽힘. **카툰 시안 금지**, 물은 대면적 예외로 채도 살짝(S~0.3) 허용.
- 운석이 물에 떨어지면: 함몰 대신 **스플래시 파티클 + 확산 링(물 반사에 잔물결)**, 크레이터 데칼 대신 일시적 파문.

### 라이트/다크 대응
- **씬 자체는 단일 톤(대낮)** — three.js 씬에 라이트/다크 모드 없음(모형은 늘 밝은 대낮).
- **UI 크롬만** 다크(§6). 씬이 밝으므로 어두운 반투명 크롬이 대비로 읽힌다. OS 다크모드와 무관하게 크롬은 고정 다크.

## 3. 라이팅

### 키라이트(태양) — DirectionalLight ×1 (그림자 캐스터)
- 방향: **좌-전방-상단**에서. 방향벡터 ≈ `dir = normalize([-0.5, 0.72, 0.5])`, 실제 position = `cityCenter + dir * (cityRadius*2.2)`. 고도 ~40°, 방위는 그림자가 **후-우측**으로 길게 떨어져 파사드 정면이 읽히도록.
- 색: `#FFF3E0` (웜 5400K). intensity **3.0** (ACES + exposure 1.05 기준. 밝지만 하이라이트 클리핑 직전).
- 그림자: `mapSize=2048`, `PCFSoftShadowMap`, `radius=3~4`(부드러운 페넘브라 — 모형은 확산광 아래라 칼그림자 아님), `bias=-0.0003`, `normalBias=0.02`. shadow.camera(ortho) 는 **도시 bbox 에 딱 맞춰** 좁게(해상도 낭비 방지).
- 그림자 농도는 **순수 검정 금지** — 아래 필/앰비언트가 채워 접지부만 진하게(AO 가 담당).

### 필/앰비언트
- **HemisphereLight**: sky `#CFE0EA`(쿨), ground `#B8AE9E`(웜), intensity **0.55**. → 그림자에 하늘 반사 채움, 웜/쿨 분리로 깊이.
- **drei `<Environment>`**: 부드러운 주간 HDRI(권장 preset 감: `park` 또는 소프트 스카이 커스텀. `studio`처럼 인공적인 건 지양). `environmentIntensity ≈ 0.7`. 반사/스펙큘러 공급용, 배경으로는 쓰지 않음(배경은 아래 그라디언트).
- **toneMapping**: `ACESFilmicToneMapping`, `toneMappingExposure = 1.05`. `outputColorSpace = SRGB`.

### 하늘 / 배경 / 안개
- 배경: **평평한 소프트 세로 그라디언트**(상 `#D6E0E6` → 지평 `#ECE4D6`). 스튜디오 배경지 느낌 — "책상 위 모형" 강화. **드라마틱 스카이/구름/god ray/렌즈플레어 전면 금지.**
- 안개: `THREE.Fog(#E7E0D2, near=cityRadius*1.3, far=cityRadius*3.2)`. **베이스 판 가장자리를 배경에 녹여** 미니어처 피사계심도 + 무한대 아닌 유한 모형 느낌. 밀도 과하면 뿌예지니 far 를 넉넉히.
- 픽셀비: `dpr=[1, 2]`, `antialias` on(또는 SMAA). 미니어처는 **엣지가 또렷**해야 정교해 보인다.

## 4. 운석 & 파괴 이펙트 (고퀄 체감의 핵심 — 가장 구체적으로)

### 4-1. 운석 4종 시각 차별화 (실루엣·머티리얼·발광·트레일 모두 다르게)

| 프리셋 | 지오/실루엣 | 바디 색·머티리얼 | 발광/열 | 트레일 | 임팩트 개성 |
|---|---|---|---|---|---|
| **rocky** (기본) | 저폴리 icosphere + 노이즈 변위, 둥글넙적 | 현무암 `#3E3833`, rough 0.85, metal 0 | 진입면 얇은 열엣지 `#FF7A3C` (약) | 얇은 회색 연기 `#8A857C` | 표준 크레이터·표준 블래스트. 웜톤 섬광 |
| **iron** (철운석) | 더 작고 조밀, 매끈-울퉁 | 다크메탈 `#5A5652`, **metal 0.9 / rough 0.35(새틴, 크롬 아님)** | **균열 사이 오렌지 용융 `#FF5A1E`, 선단 백열 `#FFD08A`** | 밝은 트레일 + 스파크 점입자 `#FFB060` | **깊고 어두운** 크레이터, 파편 임펄스↑, 섬광 백열 |
| **jagged** (파쇄형) | 각진 다면체(flat normal, 날카로운 엣지), 비대칭 | 스톤 `#4C453F`(쿨), 노출광물 밝은 면 `#7A7269` | 거의 없음(형태로 승부) | 먼지 많고 **비대칭** 흩뿌림 | 블래스트 비대칭, 파편 방향 산발 |
| **comet** (혜성) | 크고 더러운 얼음, 약한 반투명감 | 더티아이스 `#C4D2D6`(rim light+약 emissive `#DDEAF0`), rough 0.4, 암석패치 `#6E6A66` | **밝은 코마+긴 꼬리**: 코어 `#EAF6FF` → 쿨블루 `#8FB8D8` 페이드 | 리본/파티클 롱테일, 먼지 최대(쿨 `#D8E2E6`) | **넓고 얕은** 광역 블래스트, **섬광 최대·Bloom 최대**, 프로스티드 크레이터 |

- 공통: 저폴리지만 **AO/음영이 크레바스에 앉아** 싸구려 매끈함을 피한다. 크롬 미러 금지(iron 도 새틴까지만).

### 4-2. 조준/프리뷰 스타일 (기술적·절제 — 게임 HUD 남발 금지)
- **지면 레티클**: 얇고 또렷한 링(스크린 1.5~2px 감), `#FFFFFF` @0.6. 카메라 포커스 인디케이터 같은 **코너 틱/십자** 절제. 발광 네온 아님.
- **예상 크레이터 원**: 브랜드 엠버 `#F26A2E` @0.35 의 얇은 링 1~2개(R1/R2 반경). 안쪽 아주 옅은 fill @0.06.
- **수직 가이드라인**: 지면점→상공 고스트까지 얇은 **점선** `#FFFFFF` @0.3.
- **고스트 운석**: 낙하고도에 선택 운석 반투명(alpha 0.3) 프리뷰, 천천히 자전.
- **궤적**: 낙하 지점까지 작은 **점선 포물선**, 뒤로 갈수록 페이드. 화살표/두꺼운 빔 금지.
- 물 위 조준 시: 레티클 색을 쿨(`#9FD0DE` @0.5)로 미세 전환 → "여긴 물" 피드백.

### 4-3. 충돌 순간 이펙트 (스크린샷 머니샷)
타임라인(정상속도 기준): **섬광(2~4프레임) → 충격파 링 확장(0.3~0.6s) → 먼지 기둥 상승(1~2s) → 파편 정착**.

- **섬광(flash)**: 임팩트점에 밝은 emissive 스프라이트/구. 색 = 운석별(rocky `#FFF0D0` / iron 백열 `#FFF6E8`+오렌지 엣지 / comet 쿨 `#F0FAFF`). **크기보다 강도** — 작지만 강렬. ramp 1프레임 → decay 150~250ms.
- **Bloom 스파이크**: 섬광 순간 Bloom `intensity`를 베이스 0.5 → **1.4~1.6** 로 튀겼다가 lerp 복귀, `luminanceThreshold`도 0.85→0.7 살짝 낮춰 섬광만 강하게 번지게. **전체 씬 발광 아님**(임계 유지).
- **충격파 링**: 지면 밀착 확장 링, 셰이더 **반경 스케일 + 방사 알파 폴오프**. 색 = **웜더스트 화이트 `#F5EAD8`** 선단(살짝 흙빛 틴트). **사이파이 시안/퍼플 링 절대 금지.** 빠르게 확장 후 ease-out 소멸. 통과하는 건물 **창문 emissive 순간 점멸**(임팩트 반응). 부가로 아주 옅은 응결 링 1겹(폴리시).
- **먼지/연기 파티클**: 소프트엣지 빌보드(Points/InstancedMesh 풀). 지면 근처 웜그레이-브라운 `#9A8F7E` → 상승하며 밝아지고 페이드. rocky/iron 웜·어둡게, comet 쿨 `#C8CED0`·다량. 약간 상향 드리프트 + 확산. **하드 서클 파티클/컨페티 금지.**
- **파편 발광 냉각**: 코어존 큰 청크만 emissive 뜨겁게 시작(iron `#FF6A20` / rocky `#C24A20`) → 1~2s 에 베이스 머티리얼로 감쇠. 열이 식는 손맛.
- **크레이터 + 그을음 데칼**: 지면 투영 데칼. 크레이터 = 중심 함몰 `#2E2A26` + **살짝 밝은 융기 림 `#6E6152`**(방사 그라디언트). 그을음 = 소프트엣지 블랙 링 `#3A332C` 알파 폴오프, 데칼 아래 레이어. iron=더 깊고 어둡게+중심 용융 오렌지(페이드 emissive), comet=더 넓고 밝은 프로스티드 링. **다중 임팩트 데칼 오버랩 허용**(누적 파괴감).
- **카메라 셰이크**: 트라우마 기반 감쇠(회전 위주 + 미세 위치). 진폭 = 운석 크기·근접도 비례. **과셰이크 금지**(멀미 + 캡처 망침).

### 4-4. 슬로모 연출
- 물리 `timeScale=0.15`. 파편 아크·먼지·링이 슬로우로 → **DOF + Bloom + 공중 정지 파편**이 뷰티샷.
- **모션블러 OFF 유지**(캡처 뭉갬 방지 — 슬로모의 핵심은 선명한 프레임).
- 슬로모 진입 시: **아주 미세한 exposure 리프트(+0.05) + 비네트 소폭 강화**로 "집중" 신호. 색 그레이드는 중립 유지(틴트로 티내지 않기).

## 5. PostFX 스펙 (과하면 촌스럽다 — 절제 기준 포함)

권장 스택 순서: **AO → TiltShift → Bloom → 컬러그레이드(Hue/Sat, BrightnessContrast) → (미세)ChromaticAberration → Vignette → SMAA/ToneMapping.**

| 이펙트 | 파라미터(시작점) | 절제 기준 |
|---|---|---|
| **TiltShift2** (시그니처) | `blur≈0.25`, `taper≈0.4`, 샤프밴드 세로 중심 ~0.55, 밴드폭 중간. 조준/임팩트로 밴드 easing 이동 | 밴드 너무 좁으면 어지러움. 도시 핵심은 항상 샤프 유지 |
| **Bloom** | `intensity=0.5`, `luminanceThreshold=0.85`, `mipmapBlur=on`, `radius=0.6`. 임팩트 시 intensity→1.4~1.6 스파이크 후 복귀 | 임계 낮춰 **전체 발광 금지**. 창문·물반사·섬광만 번지게 |
| **AO** (N8AO 권장) | `intensity≈1.0`, `aoRadius≈0.5~0.8(월드)`, `distanceFalloff` 튜닝, 색은 **약한 웜다크**(순검정 헤일로 금지) | 접지/창틈만. 검은 후광 나오면 radius↓. **미니어처 정교함의 핵심 — 절대 빼지 말 것** |
| **컬러그레이드** | `HueSaturation: saturation +0.08` (장난감 생기), `BrightnessContrast: contrast +0.05`. 하이라이트 살짝 웜/섀도 살짝 쿨(스플릿톤 미세) | teal-orange 블록버스터 크러시 금지. +0.08 이상 채도 금지 |
| **Vignette** | `offset≈0.35`, `darkness≈0.5`, 웜다크 | 프레임 집중용. 과하면 터널. 임팩트 시 소폭 펄스 |
| **ChromaticAberration** | `offset≈0.0008~0.0012` (엣지만). 임팩트 시 잠깐 펄스 | 상시 강한 CA = 싸구려. **아주 미세하게만** |
| **AA** | **SMAA** | 미니어처는 엣지 또렷이. EffectComposer 라 MSAA 대신 SMAA |

**전면 금지**: 모션블러(캡처 뭉갬), god ray/볼류메트릭 빔, 렌즈플레어, 스캔라인/글리치, 강한 CA, 무지개 LUT, HDR 화이트 클리핑.

## 7. 이 앱 전용 Do / Don't (AI티·싸구려 3D 박제)

### Do
- **AO/contact shadow 반드시** — 건물이 지면에 딱 붙어 그림자로 접지. 이게 "떠 있는 AI 렌더" vs "정교한 모형"을 가른다.
- **저채도 대면적 + 소면적 고채도** — 파사드/도로/지면은 채도 눌러 조화, 엠버·발광·물만 채도 허용. 한 패밀리 내 미세 시드 지터로 변주.
- **유한 디오라마 베이스 + 안개 융합** — 모형판 가장자리를 배경에 녹여 "책상 위 모형" 착시. 틸트시프트 밴드는 늘 도시 핵심에.

### Don't (여기 "AI 티" 구체 박제)
- **과채도 네온**: 무지개 건물, 방사능 그린 나무, 카툰 시안 강물, 사이파이 퍼플/시안 충격파 링 → 전부 금지. 충격파는 웜더스트 화이트.
- **플라스틱/크롬 반사**: metalness 1 + roughness 0 도배, 유리=완전거울. 젖은 플라스틱 룩. → 유리 rough 0.18/새틴, iron 도 rough 0.35 새틴까지.
- **그림자·AO 없는 균일 평면광 + 전체 Bloom 발광 + 강한 CA/모션블러 + 에픽 골든아워 하늘/god ray** → 전형적 AI 3D. 우리는 **밝은 대낮 + 좁은 소프트 그림자 + 임계 높은 Bloom + 미세 CA + 평평한 스튜디오 배경**으로 간다.

## 9. 구현 튜닝 노트 (developer 반영 — 라이브 피드백 후)
> 원안 수치는 위가 계약. 아래는 "뿌옇게 씻겨나감" 피드백을 잡으며 실제 적용한 값·추가 요소. 방향(§7 Don't)은 유지.
- **TiltShift2**: 기본값이 *세로* 밴드라 도시 좌우가 뭉갰음 → **가로 밴드**(`start=[0,0.55] end=[1,0.55]`)로 교정. `blur`는 **줌 거리 연동**(근접 0.13 → 축소 ~0.015)해 축소 시 사실상 꺼짐. `taper=0.9`(넓은 포커스).
- **Bloom**: 톤매핑 **뒤**(LDR)로 옮겨 `luminanceThreshold=0.9`, base `intensity=0.28`(엔진 base bloom 0.25). 임팩트에만 스파이크 → 평상시 도시 안 하얘짐.
- **Fog**: 원경 백화 방지로 `near=R*2.1 / far=R*4.4`(도시 본체는 또렷, 가장자리만 융합). `scene.background`에 지평색 폴백(줌아웃 검은 코너 방지). Vignette `0.4/0.32`로 완화.
- **카메라**: 도시 확대에 맞춰 기본 거리 하향, `minDistance=R*0.5 / maxDistance=R*1.9`(과축소·검은 코너 방지).
- **도시 스케일**: 그리드 `gridN 8~12`(시드 파생 유지 → 크로스디바이스 재현). 정적 지오는 패밀리 머지 유지.
- **파괴**: 블래스트 반경(R2) 내 건물은 토플이 아니라 **프랙처**(1충돌 상한 high 12/low 7). 조각 수 **크기·존 비례**(근접/고층 fine 20, 그 외 coarse 12; low 9). 전역 상한 high 340/low 130 + 페이드 despawn 유지.
- **차량·보행자**: 도로 격자를 따라 도는 **InstancedMesh 에이전트**(임퍼러티브, high 46대+64명 / low 18+24), 임팩트 반경서 튕겨나갔다 리스폰. 미니어처 톤 저채도 색.

### 리얼리티 패스 (2차 라이브 피드백 후)
- **파괴 = 실치수 복셀 청킹**: three-pinata의 유닛큐브를 건물 높이로 스케일하면 세로 슬리버가 서서 남는 문제(계약이 경고했던 실패 모드) → 건물을 **실제 치수 기준 정육면체에 가까운 청크로 복셀 분할**. 고층은 세로로 여러 층 분할되어 **덩어리 잔해로 완전 붕괴**. 각 청크에 건물 축에서 바깥으로 밀어내는 **splay 임펄스**(위로 갈수록 강함) + 낮은 마찰(0.6)로 정면 붕괴 보장. **런타임 three-pinata 제거**(의존성도 제거) — 복셀 방식이 종횡비 균형·성능·상한 모두 유리.
- **잔해 잔존(정적 베이크)**: 활성 물리 청크가 잠들면(sleep) **정적 rubble InstancedMesh(링버퍼)로 베이크**하고 강체는 반환. 전역 상한/despawn은 **활성 바디에만**(high 320/low 120), 잔해는 **rubble 상한(high 1500/low 550)**까지 바닥에 남음. 반경 밖 오버플로 건물은 물리 없이 바로 rubble 더미로 붕괴.
- **운석 임베드**: 접촉 시 운석 강체 제거 → 메시를 **접촉 지점 바로 아래 지면 크레이터에 ~40% 반쯤 묻힌 채** 정착(꼬리 숨김, emissive 냉각). 최근 10개 유지. 화면 밖으로 떨어진 운석·청크는 `y<-28/-40`에서 제거(누수 방지).
- **접촉 높이 폭발(버그 수정)**: 섬광·충격파 링·먼지를 항상 바닥(y≈0.14)이 아니라 **실제 접촉 높이**에서 발생. 건물 상단 타격 시 그 높이(예: 68)서 버스트. 크레이터/그을음 데칼은 지면(운석 임베드 지점)에 지속.
- **나무**: 폭심 반경 내 나무 인스턴스 제거(가까운 건 파편으로 날림, 나머지는 작은 잔해 무더기).
- **창문 = 하늘색 유리**: 그레이스케일 창 패턴 × 하늘색 emissive 틴트(유리 `#AECADF` int 0.5 / 콘크리트 `#9FBCD6` int 0.28)로 하늘 반사 유리 톤. 아키타입별 강도 변주, 저강도 유지(형광 남발 금지).

### 불규칙 파편 + 임팩트 연기 패스 (3차 라이브 피드백 후)
- **찌꺼기 불규칙화(형태·크기·색)** — "복셀 청킹이라 잔해가 죄다 정육면체" 피드백: 단일 박스 지오 대신 **각진 파편 프리셋 5종**(지터 박스 2종 · 납작 슬래브 · 이코사/도데카 기반 각진 암석)을 시작 시 1회 생성(꼭짓점 지터 → 유닛박스 정규화 → 플랫셰이딩 facet). 파편마다 **랜덤 변종 + 비균일 스케일 + 물리 회전 + ±14% 색 지터**로 섞어 "같은 큐브 반복" 티 제거. 변종별 InstancedMesh 1개씩(액티브 5 + 잔해 5 = 드로우콜 ~10, **물리 바디 수·상한은 불변**). 복셀 그리드는 `splitAxis`로 **불균등 셀 분할** → 작은 파편~큰 덩어리 혼재. 콜라이더는 여전히 cuboid(값싸게), 시각만 각짐. 베이크 잔해 더미(`collapseToRubble`)·나무 그루터기(`treeStump`)도 여러 각진 조각을 랜덤 회전·크기·색으로 바닥에서부터 쌓아 **무더기화**(큐브 격자 티 제거). 튜닝값은 `debrisPool.ts` 상단 변종 목록 + 지터 상수.
- **임팩트 연기 기둥** — "연기 추가로 현실감" 피드백: 먼지(빠른 웜 퍼프, 낙하)와 **별개 시스템**으로, 크레이터/파괴 지점 **접촉 높이**에서 **어두운 웜그레이 연기가 상승·확산·소멸**하는 소프트 빌보드(Points, **알파블렌드**·애디티브 아님) 풀 추가. 값은 전부 `constants.SMOKE`: 색 dark `#4C4842` / (혜성·물 위) 페일쿨 `#8B9496`, `rise 3.2`, `life 5.5s`, `hold 0.5`(피크 유지 후 롱페이드), `peakAlpha 0.46`, size `6→16`(확산). **미니어처 톤 유지** — 반투명·절제, 포토리얼 두꺼운 연기 금지(§7 Don't). **연기 예산은 파편/잔해와 분리**(티어별 풀: high 8버스트×40 / low 6×26, 오래된 것부터 재사용). 도시 재생성/리셋 시 `clearSmoke`.
  > 헤드리스 검증 주의: SwiftShader(소프트웨어 GL)는 ~3fps + 고정스텝 클램프라 sim 시간이 벽시계 대비 느리게 흘러 연기 상승/기둥 형성이 압축돼 보임. 실 GPU 60fps에서 5.5s 수명 동안 기둥으로 상승·잔류. 코드(속도 적분·수명)는 정상.
